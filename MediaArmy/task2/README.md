# Задание 2: Программирование задача 1

Имеется список однотипных сущностей в виде многоуровнего массива. Каждая
сущность имеет название, ссылку на изображение и текстовое описание.
Необходимо вывести данные на экран в виде html таблицы при помощи языка PHP.
Необходимо учитывать вероятность того, что изображения или описания может не
быть, в этом случае вывод данных не должен сломаться.
